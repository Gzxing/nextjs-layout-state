## Run it locally 🏡

First, run the development server:

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Set your own 🧭

1. Clone this repo:

```
git clone https://github.com/atilafassina/nextjs-layout-state.git {{your-directory}}
```

2. Navigate to the directory and run

```
npm i && npm run dev
```

3. Be happy 🚀
